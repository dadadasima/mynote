# HTML书写规范

```html
<!DOCTYPE html> <!-- 约束，声明 -->
<html lang="en"> <!-- html标签标识html的开始 -->
<head><!-- 一般包含，title标签，css样式，js代码 -->
    <meta charset="UTF-8"><!-- 表示使用UTF-8字符集 -->
    <title>Title</title>
</head>
<body><!-- 整个html显示的主题内容 -->

</body>
</html>
```
